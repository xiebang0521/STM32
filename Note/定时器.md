# 定时器（Timer）

**定时器中断**：定时器可以对输入的时钟进行计数，并在计数值达到设定时触发中断

**Timer:16位计数器、预分频器、自动重装寄存器**的时基单元，在72MHz计数时钟下可以实现最大（65535 * 65535/ 72000 000）= 59.65s

**功能：**1.内外时钟源选择、2.输入捕获、3.输出比较、4编码器接口、5主从触发模式（主模式触发DAC）等多种功能。
